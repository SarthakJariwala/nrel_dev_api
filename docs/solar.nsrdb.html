Data Query

Find NSRDB datasets for a location

{% raw %}

class NSRDB_DataQuery[source]

NSRDB_DataQuery(api_key=None, wkt=None, address=None, lat=None, lon=None, dataset_type=None, show_empty=False)

Returns information on the closest NSRDB datasets for a location including a set of links that can be used to download the data.

{% endraw %} {% raw %}
{% endraw %}

from nrel_dev_api import set_nrel_api_key
